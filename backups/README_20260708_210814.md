# SaaS de Chatbot WhatsApp com API Oficial da Meta

Documentacao inicial do sistema SaaS multi-tenant para atendimento e chatbot via WhatsApp Business Platform.

## Stack definida

- Frontend: React, TypeScript, Vite
- Backend: NestJS, Fastify, TypeScript
- Banco principal: PostgreSQL
- Cache e filas: Redis, BullMQ
- Tempo real: Socket.IO
- Infraestrutura: Docker
- Canal: WhatsApp Business Platform / Cloud API da Meta
